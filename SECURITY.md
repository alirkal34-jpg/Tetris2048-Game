# Security Policy

## Supported Versions

This project currently supports security fixes for the latest code on the default branch.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public issue with exploit details.
2. Use GitHub Security Advisories if available for this repository.
3. If Security Advisories are unavailable, open an issue with minimal details and request private contact.

We will investigate reports as quickly as possible and provide updates when remediation is available.
